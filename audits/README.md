# Auditorias e Checklists de IA Responsável

Esta pasta contém checklists detalhados para cada dimensão de governança de IA em nosso toy example, incluindo:

- Justiça / Fairness
- Transparência
- Explicabilidade
- Auditoria e Rastreabilidade

Cada checklist está em arquivo Markdown separado, pronto para ser usado como guia de implementação e documentação.
